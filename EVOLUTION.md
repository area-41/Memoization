# Evolução Técnica do Projeto

## Etapa 1: Implementação Base
- Implementação da recursão simples para Fibonacci (Complexidade O(2^n)).
- Identificação de gargalos computacionais para n > 30.

## Etapa 2: Otimização com Memoization
- Introdução do decorador `@lru_cache` para persistência de estados.
- Redução da complexidade para O(n).
